# QuantMaster - Statistics & Quantitative Management Website

This is a comprehensive educational website for Statistics and Quantitative Management, designed for MBA students and professionals.

## Features
- **Interactive Calculators:** Scientific calculator, statistical distributions, and more.
- **Auto Graph Generator:** NLP-powered graph creation.
- **Educational Content:** Covers basics to advanced topics (ANOVA, Regression, etc.).
- **Responsive Design:** Works on desktop and mobile.
